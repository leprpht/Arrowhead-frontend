# Arrowhead

Arrowhead is a web application that provides insights into the UK's energy mix, with forecasts up to 2 days ahead. It also calculates the optimal charging time window when clean energy production is at its peak, helping users make more sustainable energy choices.

## Features

- View real-time and forecasted data about the UK's energy mix
- Calculate the optimal charging window for electric vehicles (1–6 hours) based on clean energy availability
- Intuitive card-based interface for quick insights

## Technologies Used

- **Frontend:** React
- **Build Tool:** Vite

## Usage

- Browse the energy mix cards to see current and forecasted energy production.
- Use the optimal charging calculator to find the best 1–6 hour window for charging your devices or EV when clean energy is highest.
